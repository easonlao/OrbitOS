# 我有一剑，斩灭迷茫

第四章 · 破执 见势 定心

> * "这一章，不是给你答案，是给你一把 **斩破世界观的剑**。"

这一章我只做三件事。

第一，告诉你迷茫的本质是阶段，他不是朋友，更不可能成为敌人，他如同四季里的冬。

第二，事物发展必经的六个阶段，起始期，发展期，对抗期，稳定期，衰退期，崩溃期。这是世界的运作大规律，万事万物皆在此规律之中运作，分化成为天选六段。

第三，在规律中如何用人选三剑，破执，见势，定心。找到自己的位置，自己的路，自己的心。

**"草木初生，曲卷未伸"**

我一直很喜欢冬天，更喜欢在冬天烤火取暖。没有真正的冷，我们根本不知道什么叫暖，人也是这样，很多东西，你在顺的时候是看不见的。你顺势的时候，关系，情绪，选择，这些都是轻飘飘，游刃有余，理所应当的。

很多问题，你根本不会认真看，只有当一切慢下来，冷下来，停下来。很多平时没有形状的东西，才会慢慢有形状，迷茫就此诞生。所以迷茫这件事，它不是敌人，也不是朋友，它只是阶段，它像四季里的冬天，冬天本身没有错，冷也没有错，错的从来不是冬天，是人总想跳过它。

> * 人总想只要春天，只要夏天
> * 只要生长，只要结果，只要繁茂。

可四季变化是自然法，是道的显化，不能挑选，只能观测，体会。所以"草木初生，曲卷未伸"这句话最重要的地方，是告诉你，在四季正常轮转的时候，冬天不是终点，春天也不是礼物，它们只是阶段。

所以"草木初生，曲卷未伸"，是一种人生态度和世界观，而不是一份"方法论"。也正为这样，我先把一件事说清楚。

我不能帮你们解决任何问题，我做不到，也没人做得到。我不能替你选，不能替你承受，也不能替你过完你的人生。

我能做的，只是告诉你：你们为什么总会制造这些问题，以及，怎样把制造问题的动力转化为解决问题的动力。这就是我写这一章的意义。不是让你从今以后再也不迷茫。而是让你找到，那不是天塌了，而是你又在重复制造同一种问题，这很重要。

因为人最大的痛苦，很多时候不是问题本身，而是反复制造同一种问题，却一直不知道问题是如何被自己做出来的。

> * 所以这一章，不是给你答案。
> * 是给你一把斩破世界观的剑。

很多人最大的痛苦，不是冬天来了，而是冬天来了以后，他不接受冬天是冬天，非要把冬天当成一个错误，然后开始制造更多的问题。

他开始急，开始乱，开始硬冲，开始自责，开始怀疑自己是不是废了，开始不停地找答案，不停地逼自己立刻恢复正常，这才是问题。冬天不是问题，你非要在冬天里活成夏天，这才是问题。

人间的一切事物都在人为无法控制的六个阶段中循环，我称之为天选六段：起始期发展期，对抗期，稳定期，衰退期，崩溃期。每个阶段，都有每个阶段最容易出的问题也都有每个阶段最该止住的东西：

> * **起始期·止急**——不要急着看结果
> * **发展期·止扩**——不要着急扩张，不要飘
> * **对抗期·止乱**——不要心乱，搞清战争目的
> * **稳定期·止逸**——不要懒，要一如既往
> * **衰退期·止松**——不行动，但不松懈观察
> * **崩溃期·止逆**——不要逆势，要知止

一个人的一生，一个公司，一个国家，一棵树，一片树叶，一条鱼吃另一条鱼，一栋楼，都有此过程。佛家叫无常，道家叫道，易经叫变易。这些不是结论，先记着，后面会用到。你需要在寻找势中现在找到自己的位置，这不是本章重点的内容，以后还有专门有篇幅介绍天选六段。

> **你为什么会制造问题**

很多人以为，问题是外面来的，其实不全是，很多问题，都是人自己制造出来的。不是凭空制造，而是在一个本来正常的阶段里，因为不接受这个阶段本来的样子，于是额外制造出更多问题。

比如冬天本来就该冷，可你非要让自己像夏天一样热，你就会急，急了以后就乱，乱了以后就硬做决定，硬做决定以后就出错，出错以后就自责，自责以后就更冷，这就是你自己把一个阶段，活成了一个问题。

所以后来我越来越觉得，人的痛苦常常不是来自阶段本身，而是来自对阶段的不接受，不接受自己在起始期，却想立刻有成熟期的结果，不接受事情已经进入对抗期，还觉得一切都该像发展期一样顺。

> * 不接受一段关系已经在衰退
> * 非要把它按回稳定期
> * 不接受自己现在就是迷茫
> * 非要立刻清醒

这才是制造问题的根，你不是因为迷茫而痛苦，你是因为不接受迷茫是一个阶段，于是开始在里面疯狂加戏。所以第一步永远不是解决，第一步永远是：可能的话尽量减少额外制造问题。

> **第一种迷茫：看不见路**

不是没路，是你站在那里，前面像有雾一样。不是本来就没有路，你本来有条路走得好好的，但你拿着一个望远镜，看着路的前端好像不是你想要的样子，你着急去换一条路因为那边听说很好，但你的望远镜看不穿那路上的迷雾，对未来望远镜能看到确定性的麻木悲伤，感到人生苦短没有尽力去体验。

于是走到新的路上，你问自己要不要走，你不知道;你问自己往哪边走，你也不知道。这种时候，人最容易制造的问题是什么？是贪婪地选路，导致选后的恐惧，恐惧后着急。他一着急，就想立刻有答案，立刻做选择，立刻证明自己没有浪费时间，于是问题就来了。所以第一种迷茫的核心，不是"没有方向"，而是在看得清楚方向的地方觉得失望：在看不清的时候硬要立刻看清，这就开始制造问题了。

**这时候，三步要怎么用：**

**先破执**——破执不是放下，也不是骗自己说"没关系"，破执的核心，是缩小自己，放大世界。庄子逍遥游里有一句大家都知道的话："北冥有鱼，名为鲲，其大几千里也"，讲的就是把自己缩小、把世界放大的故事。你要做的第一件事，不是找路，而是把自己从那个被放得过大的问题里抽出来。问自己：我现在到底在执什么，是执着一定要马上有答案，还是执着自己不能错，还是执着不能让别人觉得我没用？很多人不是看不见路，是他太怕承认"我选的新路错了"我的老路很好"，这就是执。

**再见势**——路已经换了，我要看看我在这条路的六个路口中的哪一个？如果你还在起始期，那你现在看不清，很正常。起始期本来就不清楚，本来就模糊，本来就还没长开。"草木初生，曲卷未伸"，它本来就不是一眼能看明白的状态，所以很多人不是没路，是拿成长期的要求去要求起始期的自己，这就是看错势。

**最后定心**——这时候你该止什么？止急。就这一条，别催自己，别逼自己，别在雾里奔跑。看不见路的时候，最好的动作不是乱走，是先别多制造一条错路。找一种安全的方式走，慢慢探路慢慢摸索，下个路口回到，雾也会散去。所以第一种迷茫，你不是去解决它，你只是别在看不见的时候，额外制造更多混乱，这就够了。

> * **第二种迷茫：路太多**

第二种迷茫，不是没路，是路太多。有好的家庭，有好的学历，到处都在邀请自己，有一些生活得很体面的积蓄。路在眼前眼花缭乱，本来没有选择余地的人生，被你过得看起来条条大路通罗马。每一条看起来都行，每一个结果都舍不得，于是人停在那里。这种时候，最容易制造出来的问题是什么？是贪。不是普通的想要，是"全都想要"。你舍不得丢掉任何一种可能，于是你一条也走不了。这时候三步怎么用？

**先破执**——先看清自己执什么。你是不是想把所有可能都留下，是不是每一条路都舍不得放，是不是每一个结果都想要一点，是不是每一个代价风险都不想承担。承认这个，路才有可能开始变清。

**再见势**——然后看阶段。现在那么多路对于我而言，我到底在哪个阶段？选择应该经历的事物周期，而不是贸然决策。如果已经进入对抗期，你还在按发展期的心态硬冲;如果已经衰退了，你还在当它刚开始，那你当然越选越乱。

**最后定心**——这时候要止什么？**止飘，也止多**。路太多的时候，最重要的不是选得漂亮，而是先舍掉一部分，不是因为那些路永远不行，而是因为你现在不该全走。所以第二种迷茫的重点，也不是解决，而是少制造一点，少制造那些"什么都不放""什么都不愿失去"的问题。

> * **第三种迷茫：既想变化，又不想变化**

第三种迷茫，是最深的。你一边想让人生变好，一边又想让某些东西永远别变。想要机会，渴望变化，又不想接受变化出自己不期待的场景。想要伴随时间成长，自己越来越强，事业越来越好，又不想伴随时间流逝，自己越来越老，皱纹越来越多。这才是最根本的矛盾。这种时候，你制造出来的问题是什么？是妄想。

妄想世界只改对你有利的那一半;妄想时间只带来好处，不带走东西;妄想自己既能一直往前，又永远不用付代价。这就是第三种迷茫最麻烦的地方。所以三步怎么用？

**先破执**——先承认，你执着的不是变化本身，你执着的是"只有好处的变化"。你不是真的想接受四季，你只是想永远停在自己喜欢的那一季。

**再见势**——然后看势，任何关系，任何事业，任何人生状态，都在起始、发展、对抗、稳定、衰退、崩溃里走。你想把它永远固定在某一个阶段，本身就是在逆着轮转。

**最后定心**——这时候该止什么？**止妄想，止不舍**，止那个"既要变化又不想变化"的心。这一步不是解决矛盾，是承认矛盾本来就在这里，你一承认，问题就少一半，因为你终于不再逼世界按你的偏好运行了。

> **所以这一章到底在讲什么**

这一章不是在教你解决迷茫，这一章是在告诉你：迷茫本身不是问题，阶段本身不是问题，冬天本身不是问题。真正的问题是，你总是在一个本来正常的阶段里，额外制造出一堆不必要的问题，在看不见路的时候制造着急，在路太多的时候制造贪多，在既想变化又不想变化的时候制造妄想。我能做的，只是把这些制造问题的方式指出来，以后你再进冬天的时候，能少制造一点，这就够了。这就构成了人选天选论的原文：

> * **人选**——非逆势而为，亦不妄图停势，只顺势而止于其所当止。
> * **天选**——非赐其欲也，而应其舍也。

最后，很多人在读人选天选论，会有一种很压抑的感受，或者很刺痛的感受，这是正常的。我举一个例子希望你们能明白我的意思。

**草木初生，曲卷未伸**

人在冬天里，把手伸进雪里，先感觉到刺骨的寒，冻得痛，再过一会，你感觉好像没有那么冷了，手开始很暖。很多人以为这是适应了，这不是暖，是血管在"冰冷诱导性扩张"。如果你继续保持，很快就没知觉了。

> 人最危险的状态不是疼，而是"不疼"。

反之如果你在冬天，找一个地方坐下来，生一团火，火不用很大，你慢慢靠过去，最开始不是暖，是刺，过一会儿，那种暖才会一点一点出来。这两种感觉，表面上很像。都是从"难受"慢慢变成"没那么难受"，但本质完全不一样。

> * 雪里的那种"暖"，是你在失去感觉。
> * 火边的那种"暖"，是你在恢复感觉。
> * 一个是在关闭。
> * 一个是在打开。
> * 一个是让你越来越迟钝。
> * 一个是让你越来越清楚。

所以人其实很容易判断错，很多时候，你以为自己"适应了"，其实你只是开始感觉不到了。而真正的恢复，从来不是一开始就舒服。真正的恢复，一开始都是刺的，是慢的，是有感觉的。是你能清楚地知道，自己在一点一点回来。

所以后来我越来越在意一件事:不是我现在难不难受，而是——我现在的"变好"，是让我更有感觉，还是让我更没感觉。

好了，我们准备走进人选天选论的选择之路上最重要的"桥"——**"天权"的诞生。**
