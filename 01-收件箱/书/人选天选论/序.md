# 序

> 本书全部原文会在网络发出

很多年前，我想把这本书包装成一本被我偶然发现的"上古残卷"。因为我不认为大家会重视一个乳臭未干的毛头小子写的关于"选择"与"代价"的思考，包括我写出"人选祸，天选福"也是为了更易于传播，于是"代价"随之而来。我和你们的故事从我为了传播而"选祸为福"开始。

我的父亲是个读书人，在他的熏陶下自幼就看了很多书。15 岁的时候我就在考虑自己要写一本书出来，当时我就思考，为什么大家都愿意去学习心经、金刚经、道德经，而只有极少的人愿意去看庄子和涅盘经，最后总结下来感觉"易于传播"是思想是否会被大众接纳的核心。心经、金刚经，字很少，背也容易背，抄写也容易抄写；道德经，"道可道，非常道"朗朗上口，易读易背；而庄子，我现在都一段话还有很多字读不出来。为了易于传播所以我提出"人选祸天选福"这朗朗上口的口头禅，导致了非常多的人对它的误解，也直接导致了我现在电脑前码字，让我的"人选天选论"早产。这里首先和被我误解的人道歉。

基于此，这本书中我也就不以文言文原文的形式直接阐述，也不着急让你们看到价值，我们就从认识我主观的世界开始，邀请大家进入我的世界。

我主观的认为，世界运行在规律之中，宇宙是决定论的。电子的跃迁、晶体的构造、星球的运动、化学反应、人类的进化，这些都不是选择的结果，而是规律的必然结果。也就是说从宇宙诞生的原点，一切就已经是一本书了，时间只是我们打开书本翻页阅读的顺序，时间只是测量变化的现象的尺子。人类的思想也是，神经反应、基因，都只是规律的显化。

> * **严格意义上来说在唯物的世界里：人也是没有自由意志的。**

包括我正在这里写人选天选的序言，和你正看着这些难以理解的文字，这一刻都是注定的。

但深刻的思考之后，人必须"假装"自己有选择的权利。因为如果你真的相信一切都注定

> * **那为什么要努力**
> * **为什么还要放弃**
> * **为什么还要选择**

所以我们必须生活在一种简单的状态中，明知一切可能被决定，但仍然以"选择者"的身份去行动。就如生而必死，我们也不会放弃生存。

很多人误解说人选祸天选福，是命理。其实我是不认为未来可以预测和干涉的，这里会聊到我的 **"破画原理"**：画中人无论在画中力气多大，他是不能撕毁画卷的。用科学的角度去解释，就是观测本身就是变量的一部分。

> * **这是我这颗硬币的正面：唯物世界。**
> * **下面我们去看看我的反面：唯心世界。**

在我的心中，任何物体都不真实存在于这个世界中，它只是事物在心中的投影。举例说明，你正在看人选天选论，你大概知道我还活着，我什么样子，也大概知道我晚上还要直播，这些都是存在于你心中的，不是真实存在于这个世界的。比如晚上直播前我被车撞死了，自此以后再也没有人会继续更新，我慢慢就被大家遗忘了，但当偶尔你的一次选择时想起了我，那你可能会猜测我隐居了，对世界失望了，那时我活在你心中，直到你有一天看到新闻"著名造船博主路飞车祸身亡"，那一刻我死了，死在你心中。当然这一切也可能都是我自导自演的故事，我正在南极看企鹅。当有一天你看到我在南极滑雪的视频，你很开心，我又活在你的心中；你认为新闻错了，当看到右下角"视频疑似 AI"，我又在你心中死了一次，可能 AI 的标识是我自己 PS 上去的，唯心的世界是很宏观的。唯物的世界和心学世界就是一个硬币的两面，它同时存在但无法被同时观察。

好了这一段太烧脑了，但必须要讲，后面的章节中会有具体的细节，作为序也只是为了讲清楚我这本书大概说了些什么。

由这两个主观的世界，形成了一个主观的观念，也是"人选祸天选福"希望实现的目的：人生无法决定结果，但可以选择承担什么代价。我们无法避免痛苦，也无法保证得到自己想要的。人生并不是为了得到什么或者失去什么，我们正在需要建立的是一个长期能够稳定生存的结构，可以类似理解成我自己一直追求的善终，可能不准确预言的表达就是这样，好理解就不准确了。

> * **A 赚了很多钱，但每天焦虑、恐惧、失眠，他的"得到"很多，但结构是不稳定的。**
> * **B 收入普通，但生活秩序清晰、关系稳定，他的"得到"不多，但结构稳定。**

这是个糟糕的例子，但我想表达，我认为稳定的结构，优先于一切。这些年，我见过很多迷茫和痛苦的人。有的贫穷，有的财务自由；有的读过很多书，有的做过很多事；有些人很好看，有些人很丑。但他们都很痛苦。其实通过观察我更确定，痛苦根本不是某一类型人的专利，而是所有人必须面对的——众生皆苦。

痛苦的核心原因就两个大类：

> * **"得不到自己想要的"**
> * **"已失去自己没珍惜的"**

16 岁那年，我和文杰偷拿了父母的钱跑去神龙架旅游，木鱼村很美，我们在小溪边散步。溪水很清，石头看起来很特别。可当我蹲下来看，看得到水，却看不清石头。水流很急，水从石头上翻过去，水面不断起伏、反光、扭曲。

水看起来很浅，我想用手去拿河底的石头，手伸下去一半，马上缩回来，水真的很冷，那一刻感觉思想都被冻结了。第二天我们又来溪边游玩，文杰见我怕冷，他去拿水底的石头给我。看到他拿起来的一刻，石头并不是我想象中的那么美。

> * **贪婪——想拿美丽的石头。**
> * **恐惧——感受到寒冷退缩。**
> * **已失去——石头神秘的美感。**
> * **得不到——心中想要的石头。**

思索立刻在我心中环绕，我要求文杰暂时不要打扰我，开始思考这期间的关系，写进日记。这个日记很长，讲了河流和知识的关系、思想速度以及思考效率、观察自己内心贪婪恐惧的方法、借口、自洽等等。

我把人心中的贪婪和恐惧比作河底的一白一黑两种石头，把借口和自洽比作被石头搞的乱七八糟的表面水流。贪婪让人伸手，恐惧让人缩手。想得到，是贪婪；怕失去，是恐惧。

人一生的很多决定，其实都在这两块石头之间，但问题从来不在贪婪，也不在恐惧。问题在于——这两者之间，还有没有可以选择的结构，这就是天权。如果贪婪与恐惧还能互相牵制，人就还能稳住。如果其中一边彻底压倒另一边，人就会走向极端，陷入病态的抑郁和焦虑。极少的人只剩贪婪，只剩恐惧；大部分的人，在两者之间摇摆。

于是我主观的把人的状态简单分成五种境界：

> * **迷失**——天权崩塌，一路加码一路退缩。
> * **求福**——天权摇摆，趋利避害。
> * **见机**——天权可视，清晰地看到自己的：选择、代价、结果。
> * **持衡**——天权可控，明白人生无法避免代价，主动承担选祸。
> * **两忘**——天权消散，没有贪婪，没有恐惧，故两相忘，不需要天权。

但人并不会永远停在某个境界里。人心似水，随境而动。一个人今天清醒，明天也可能愚蠢。刚刚看透，下一秒又可能被自己的欲望拖走。所以这些境界只是即时状态。我知道你们现在先在给自己看自己在哪个境界，也在思考姜蓝在哪个境界。后面正文会讲，这里只能说，我不敢也没资格走进两忘，目前我也怀疑是否有人真的能走到两忘？

这和我一直以来的说的，拿最大的勾、最大的饵、吊的最大的鱼有关。大多人求福。哪里有好处，就往哪里走；哪里有风险，就立刻退开。他们看起来很聪明。但在我主观的世界里，世界从来不奖励谁。世界只收代价。你做任何选择，都要付代价。有的立刻支付，有的很多年以后才结算。

很多人相信祸和福之间的因果有一张账单。忍一忍，将来会有回报；吃点亏，命运会补偿。但世界不是银行。祸和福之间，没有可以回头核对的账本。于是我后来把很多事情总结成 3 个词：

> * **人选/祸/天选**

人选，是你当下的选择。在人选的世界里只有人和天，人是自己，天是除了自己以外的变量。

祸，不是灾难，祸是代价。

天选，是结果。你可以决定选择，但结果从来不只属于你一个人。大家会很疑惑，福去哪里了？福是稳定的结构，这个正文讲。

很多人喜欢用一个词解释这一切——**"运气"**。是因为"运气"太容易让人停止思考、自洽。在我看来，很多所谓运气，不过是你看不见的选择叠加。你能决定的就是你的选择，其他除了你以外的一切事物的选择皆为天选，天选不可测。无数选择交织在一起，最后形成结果，你看不见全部路径，于是把它叫作运气。所以在这本书里，我会反复讲一句很主观的话：这个世界没有运气，只有选择。当然，如果再往深处走，就会遇到一个更大的问题。

如果宇宙的一切都在规律之中，如果人的意识也可能只是规则的一部分，那么选择到底算什么？对此我并不回避。

甚至我愿意承认：我们也许真的只是画中的人。这里还要说到"破画"。你无法跳出"命运"，但你仍然必须为自己的这一生负责。很多人一生都在试图避免痛苦，可他们不知道，痛苦无法避免，快乐与痛苦，绝对是一半一半。你唯一能决定的，是你愿意用什么方式承担。所以这本书并不是教人成功，也不是教人得"福"，不是为了得到什么，也不是为了避免失去什么，而是为了建立一种结构——一种即使世界不断变化，也不会轻易崩塌的结构。

人无法追求永恒，因为永恒本身就是幻觉。真正稳定的东西，从来不是不变，而是能够承受变化。如果这本书真的有一点价值，我希望它不是让你赢过谁，而是当你再一次站在那条河边的时候，你能比过去更清楚一点：

> * **你为什么伸手**
> * **你为什么退后**
> * **你为什么选择**
> * **你为什么承担**

如果有一天，你终于能看见自己河底的那两块石头，那这本书，大概就没有白写。

好了，亲爱的读者，我们精神世界的对碰，就从这里开始。

## 附

**《人选天选论》**：本书全部原文会在网络发出，主要讲解作者本人思想的宇宙观，世界观，和价值观。

**《人选天选人》**：主要是描述，如何通过学习人们选择中贪婪和代价的符号，建立天机，概率性的判断和预测一个人和事物的内心世界，但代价巨大，通过天机观测天权，你会发现，天权在对抗和分裂统一的循环态往复，而我们希望改变环境和结构的真正机会，就是天权统一和天权分裂的关键节点，这时人选天选的反馈效率特别高，在天权结构稳定的时候，没人能通过自己改变结构，无论你多努力反馈效率都很低，但切记反馈效率提高和我们期待的机会本身，只能用于建立更加稳定结构，调节天权，否则会在在结果中付出巨大的代价。

**《人选天选天》**：主要描述，探索寻找因果账单观察者欺骗的思考，探寻走向两忘的设想以及窥探的唯心假设。
