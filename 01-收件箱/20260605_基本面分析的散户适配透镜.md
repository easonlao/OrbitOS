# 基本面分析的散户适配透镜

> 从专业基本面分析中提取散户可用的简化框架。核心理念：**用基本面定方向，用技术/结构定时机**。

## 散户的核心约束

1. **时间**：东八区，美盘凌晨，无法盯盘
2. **信息**：无 Bloomberg 终端，财报有 1-3 月滞后
3. **资金**：分散化能力有限，深度调研单个标的能力有限
4. **知识**：非专业会计背景，复杂 DCF 建模不现实

## 三指标快速筛

放弃复杂绝对估值，用三个相对指标做初筛：

| 指标 | 公式 | 健康区间 | 说明 |
|------|------|---------|------|
| **PEG** | PE / 盈利增长率×100 | < 1（低估），< 0.5（极便宜） | 彼得·林奇经典，散户友好 |
| **ROE** | 净利润 / 股东权益 | > 15%（优秀），> 10%（合格） | 资本效率，高 ROE+低负债 = 优质 |
| **负债权益比** | 总负债 / 股东权益 | < 1（稳健），< 0.5（保守） | 排除高杠杆雷区 |

三个指标同时满足才能进入关注名单。不追求最佳估值，追求避开明显高估和财务地雷。

## 能力圈原则

**只做理解的行业和商业模式**。巴菲特的能力圈概念是散户基本面分析中最重要的防御机制：
- 如果无法用三句话讲清这家公司怎么赚钱 → 不在圈内
- 如果行业技术路径无法判断 → 不在圈内
- 如果财报有看不懂的科目 → 先学懂，或者跳过

能力圈可以逐步扩展，但每次只扩展一个行业。

## 信息源建议

- **财报**：SEC EDGAR（免费）、公司 IR 页面
- **宏观**：Trading Economics、美联储官网
- **行业**：Seeking Alpha（行业分析长文）、券商研报摘要
- **筛选**：Finviz（免费股票筛选器）、TradingView 基本面面板

## 使用节奏

- **每季度**：更新关注列表的 PEG/ROE/负债率
- **每月**：浏览持仓公司新闻和行业动态
- **每周**：宏观事件日历（CPI、FOMC 等）
- **不入圈的不分析，不分析的不持仓**
