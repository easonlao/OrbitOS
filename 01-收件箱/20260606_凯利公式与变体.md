# 凯利公式

## 公式

```
f* = (b·p - q) / b
```

- f* = 最优下注比例（占本金百分比）
- p = 胜率, q = 败率, b = 盈亏比

## 核心约束

胜率和盈亏比为输入参数，实盘中两者都不稳定。持有时间（第三个变量）将两者锚定：时间越长，胜率越低，盈亏比越高。

## 变体

| 变体 | 比例 | 适用 |
|------|------|------|
| 全凯利 | 1× f* | 理论基准 |
| 半凯利 | 0.5× f* | 机构量化 |
| 四分之一凯利 | 0.25× f* | 散户实盘 |
