# 技术指标

技术指标是对价格、成交量等市场数据做数学运算后绘制的辅助线或数值，用于识别趋势方向、动量强弱、超买超卖和潜在反转信号。

## 分类

| 类型 | 功能 | 代表指标 |
|------|------|----------|
| 趋势类 | 识别方向、过滤噪音 | [[#移动平均线 MA]]、ADX |
| 动量类 | 衡量涨跌速度 | [[#RSI]]、[[#MACD]]、Stochastic |
| 波动类 | 衡量价格离散度 | [[#布林带]]、ATR |
| 成交量类 | 验证资金参与度 | OBV、Volume Profile |

## 核心指标

### 移动平均线 MA

$$SMA = \frac{P_1 + P_2 + ... + P_n}{n}$$

$$EMA_t = P_t \cdot k + EMA_{t-1} \cdot (1-k), \quad k = \frac{2}{n+1}$$

- SMA: 等权平滑，滞后性大，用于判断长期趋势
- EMA: 近期价格权重大，反应更快，适合动态市场
- 常见周期: 20/50/200（SMA），12/26（EMA）

### RSI

$$RSI = 100 - \frac{100}{1 + RS}, \quad RS = \frac{\text{Avg Gain}(n)}{\text{Avg Loss}(n)}$$

- 取值 0~100，默认周期 14
- > 70 超买，< 30 超卖
- **背离信号比超买超卖阈值更可靠**: 价格新高但 RSI 未新高 → 顶背离

### MACD

$$MACD = EMA_{12} - EMA_{26}$$
$$Signal = EMA_9(MACD)$$
$$Histogram = MACD - Signal$$

- 金叉: MACD 上穿 Signal → 买入信号
- 死叉: MACD 下穿 Signal → 卖出信号
- 零轴上方 = 多头主导，下方 = 空头主导
- 背离: 价格与 MACD 方向相反 → 趋势可能衰竭

### 布林带

$$Middle = SMA_{20}$$
$$Upper/Lower = Middle \pm 2\sigma$$

- 统计学基础: ±2σ 覆盖约 95% 的价格分布
- 带口收缩 → 低波动 → 即将突破（Squeeze）
- 带口扩张 → 高波动 → 趋势延续
- 价格触及上下轨不自动反转，需结合动量指标确认

## 关键原则

1. **不孤立使用**: 单一指标的假信号率高，必须多指标共振
2. **趋势 + 动量 + 波动组合**: 三类指标互补，例如 MA 定方向 + RSI 找入场 + 布林带判波动
3. **多时间框架**: 大周期定趋势，小周期找入场
4. **所有指标都是滞后的**: 基于历史数据计算，不预测未来，只辅助判断概率
