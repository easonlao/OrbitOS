# 人选天选论 - 框架笔记

## 一、根本命题

> 人生无法决定结果，但可以选择承担什么代价。

目标不是成功、不是得福，而是建立一个**即使世界不断变化也不会轻易崩塌的结构**。

## 二、核心三元

| 概念 | 含义 |
|------|------|
| **人选** | 你当下的选择。可控域：只有自己的行动 |
| **祸** | 不是灾难，是**代价**。每项选择都需支付 |
| **天选** | 结果。涉及除你之外一切变量的总和，不可测 |

口头禅：「人选祸，天选福」

## 三、河底石头模型

- **贪婪（白石）**：想要伸手去拿
- **恐惧（黑石）**：感受到寒冷退缩
- **水面湍流**：借口与自洽，被石头搅乱

核心洞察：问题不在贪婪，也不在恐惧。问题在于**两者之间还有没有可选择的结构**。

## 四、天权：选择的结构空间

**天权** = 贪与惧之间尚存的、可供选择的空间。

- 贪惧互相牵制 → 人能稳住
- 一边压倒另一边 → 走向极端（病态焦虑/抑郁）

### 五种境界

1. **迷失**：天权崩塌，一边完全压倒另一边
2. **求福**：天权摇摆，趋利避害（大多数人所在）
3. **见机**：天权可视，清晰看到选择/代价/结果
4. **持衡**：天权可控，主动承担代价
5. **两忘**：天权消散，无贪无惧（作者自认未达）

人不会停在某个境界，人心似水，随境而动。

## 五、四组选择配置

| 情境 | 行为 | 结果 |
|------|------|------|
| 在贪婪中 | 拿起贪婪 | **福** |
| 在贪婪中 | 放下贪婪 | **祸** |
| 在恐惧中 | 放下恐惧 | **福** |
| 在恐惧中 | 拿起恐惧 | **祸** |

逆直觉法则：
- **顺着本能选**（贪时拿、惧时退）→ 短期舒服 → 天选祸
- **逆着本能选**（贪时放、惧时进）→ 短期难受 → 天选福

## 六、天机：洞察他人选择的符号系统

通过观察一个人选择中贪婪与代价的符号，概率性地判断其内心世界。**代价巨大**。

天权在**统一与分裂**的关键节点时，反馈效率最高——这是改变结构和环境的窗口。但反馈效率只能用于**建立更稳定结构**，否则代价极大。

## 七、前提预设

1. **唯物面**：宇宙是决定论的，人无自由意志，但必须「假装」有选择权去行动
2. **唯心面**：事物只存在于心中的投影，唯物与唯心是硬币两面，不可同时观测
3. **破画原理**：画中人无法撕毁画卷；观测本身就是变量的一部分

## 八、局限（作者自述）

- 只能解决「不失去已有的」，无法解决「得到未有的」
- 不是成功学，不是算命，不预测具体结果
- 天选不可测：所谓运气，只是你看不见的选择叠加

## 九、与交易框架的可能关联

轩哥关注的方向：

1. **选择与代价的结构化**：交易决策本质是承担有限代价、获取概率优势——与人选天选论中「主动承担代价」一脉相通
2. **天权窗口**：天权统一/分裂节点反馈效率最高 → 类比市场关键转折点（order flow imbalance？）
3. **反直觉操作**：「贪时放、惧时进」←→ 交易中追涨杀跌是本能，逆向是专业
4. **稳定结构优先于结果**：交易系统的核心不是单次盈亏，而是长期可重复的结构
