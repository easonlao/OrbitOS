# MMXM（Market Maker Model）做市商模型

## 定义

MMXM 是 ICT 描述做市商如何完成价格交付的四阶段模型：原始盘整 → 工程化流动性 → 智能资金反转 → 趋势分配。

## 四个阶段

1. **原始盘整（OA）**：价格横盘建立区间，高/低点成为后续流动性目标
2. **工程化流动性（Judas Swing）**：突破区间一侧，扫掉散户止损，获取对手盘
3. **智能资金反转（MSS）**：价格逆转，突破区间另一侧，确认方向
4. **趋势分配**：朝目标流动性池运行

## 两种路径

- **买入模型**：盘整 → 向下扫 SSL → MSS → 看涨 FVG → 向上到 BSL
- **卖出模型**：盘整 → 向上扫 BSL → MSS → 看跌 FVG → 向下到 SSL
