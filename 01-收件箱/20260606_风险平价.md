# 风险平价

## 定义

风险平价（Risk Parity）使组合中每个资产的风险贡献相等，而非资金分配相等。

## 与传统等权重的区别

等权重：各 50% 资金 → 高波动资产贡献 90% 风险。风险平价：调整资金权重 → 各资产风险贡献均匀化。

## 核心公式

```
资金权重 ∝ 1 / 波动率
```

## 代表策略

桥水全天候（All Weather）：覆盖四个经济象限（增长↑/↓ × 通胀↑/↓），每个象限有对应的保护资产。
