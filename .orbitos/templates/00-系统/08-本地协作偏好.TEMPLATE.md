# 本地协作偏好

> TODO_UPDATE_LOCAL_COLLAB_PREFS: 这是首次初始化生成的本地模板。请按你自己的真实习惯替换下面的示例内容；未更新前，Agent 不应把它当成稳定偏好。

本页只记录当前 Runtime 使用者与 Agent 的稳定协作偏好。它帮助 Agent 更快进入合适的合作方式，但不自动代表其他 Runtime，也不自动进入 Product Repo 的通用规则。

## 默认合作方式

1. 方向通常由你来定，Agent 负责收敛落地。
2. 常规任务默认直接推进；有明显分叉时再先对齐。
3. 输出优先给结论、改动、验证结果和剩余事项。
4. 如果范围膨胀、边界混乱或验证不足，Agent 应直接提醒。
5. 稳定结论可以沉淀；临时讨论先不要急着升级成系统规则。

## 可按需补充

- 你希望 Agent 更主动，还是更保守？
- 你更偏好短结论，还是带一点上下文？
- 哪些事情必须先问你，哪些事情可以默认直接做？
- 你最不喜欢哪类输出或协作方式？
