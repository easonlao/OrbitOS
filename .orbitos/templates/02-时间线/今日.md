# 今日

> 日期：首次使用时由 Progress Sync 更新

## 1. 今日总览

### 当前判断

- 这是 OrbitOS 的每日 Dashboard 模板。
- 第一次使用时，先让 agent 执行 Startup Sync。
- 如果 agent 尚未注册，它应停止并请求你确认 `agent_id` 和部署信息。
- 完成首个 onboarding 或实质性工作后，再由 Progress Sync 刷新本页。

### 系统健康

- 发布模板不预置真实 agent、局域网 IP、定时任务或 Hindsight bank。
- 本页不手动堆积长期项目状态；长期状态进入项目 `STATUS.md`，本页只投影当天关键事项。

## 4. 可继续

### 推荐继续

- 注册第一个 agent。
- 把材料放入 [[../01-收件箱/00-粘贴|00-粘贴]] 或 `01-收件箱/`。
- 让 agent 执行 Progress Sync，刷新当前状态。

### 可并行推进

- 运行 validation：`python .orbitos/scripts/run-validation.py`。
- 根据需要接入 Hindsight，但 Hindsight 不是 OrbitOS 必需项。

### 暂不推进

- 不要在未确认前创建正式知识卡片、ADR 或长期规则。
