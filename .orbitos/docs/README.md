# OrbitOS Design Docs

This directory contains design and explanation documents for OrbitOS internals.

Use this directory for:

- requirements
- architecture notes
- design rationale
- unresolved tradeoffs
- implementation plans
- schema and workflow design explanations

Do not put stable execution rules here.

Stable agent rules belong in `.orbitos/rules/core/`.

Learned, evidence-tracked rules belong in `.orbitos/rules/learned/`.
