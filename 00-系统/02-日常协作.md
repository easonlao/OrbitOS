# 日常协作

你可以像交代同事一样向 Agent 提出任务。OrbitOS 会把结果放回合适的位置，让下一次继续时不必重新解释背景。

## 提出任务

直接描述你想要的结果即可：

```text
整理收件箱里最近新增的内容，先给建议，不要移动。
继续推进这个项目，告诉我现在做到哪里。
把这份材料整理成知识草稿，先不要定稿。
今天还有什么需要我确认？
```

你可以随时用一句补充限制范围：`只看这个项目`、`先不要修改`、`先写草稿`、`先告诉我影响`。

## 一次任务结束后，你会看到什么

完成后，结果会回到最适合的位置：

- 项目进展回到对应项目。
- 已确认的长期结论进入知识区；未确认内容保留为草稿。
- 今日需要关注的变化汇总到 `02-时间线/今日.md`。
- 需要你拍板的事会明确说明原因和选项。

你不需要追踪 Agent 如何处理；只需要检查结果是否符合预期。想让当前视图刷新时，说“同步进度”即可。

## 复杂问题怎么处理

当问题模糊、需要权衡或你不确定从哪里开始时，说：

```text
启动思考模式
```

Agent 会先回顾当前讨论，告诉你这一步更适合从哪些角度思考，并解释推荐理由。你选择后才会继续；详细说明见 [[11-思考方法|思考方法]]。
