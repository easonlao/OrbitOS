# 工作日志管理工作流

## 可用内部工具

| 工具 | 用途 |
|------|------|
| `create_worklog()` | 生成今日工作日志 |
| `append_worklog(content)` | 智能追加内容到今日日志 |
| `rebuild_worklog_index()` | 重建日志索引 |

## 日志存储位置

`{VAULT}/02-日记/工作日志/YYYYMMDD_工作日志_周X.md`

## 日志模板结构

```markdown
# YYYY-MM-DD 周X

## 今日工作内容
## 重点记录（可直接粘贴）
## 问题 & 解决方案
## 参考 / 链接
## 明日计划
```

## 内容追加分类规则

| 关键词/特征 | 目标章节 |
|-------------|----------|
| 问题、bug、报错、异常 | 问题 & 解决方案 |
| 解决、修复、fix | 问题 & 解决方案（解决部分） |
| 完成、实现、开发、重点 | 重点记录 |
| 链接、参考、http | 参考 / 链接 |
| 明天、计划、待办、todo | 明日计划 |
| 其他 | 今日工作内容 |
