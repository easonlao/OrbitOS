# 系统变更

本页只记录会改变你使用方式的重要变化，不记录内部实现过程。

## 当前可用的能力

- 收件箱、项目、知识和今日页构成日常协作的核心闭环。
- 阅读、人物、Hindsight、工程、自动化与多 Agent 协作按需启用。
- 复杂任务可通过“启动思考模式”让 Agent 先匹配思考模型。
- 重要移动、删除、归档和定稿由用户确认。

## 最近变化

- 用户说明已按“功能、说法、结果”重写，Agent 内部操作不再作为日常阅读内容。
- 模块说明已按场景拆分，可直接从 [[06-模块与扩展|模块与扩展]] 进入。
- 思考方法提供显式启动入口：它会先回顾当前讨论，再推荐能改善本轮判断与产出的思考模型。
