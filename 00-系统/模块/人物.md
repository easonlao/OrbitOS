# 人物模块

人物模块帮助 Agent 逐步理解“怎样和你协作更合适”，而不是给你贴一个固定的人格标签。

## 它解决什么问题

当你发现 Agent 经常误解你的表达偏好、决策节奏、对细节的需要，或总把你不喜欢的工作方式重复一遍时，人物模块可以把这些经验变成可校准的协作依据。

它从一轮简短的基线问卷开始。MBTI 只用作低成本的起点，之后更重要的是你真实使用系统时表现出来的偏好和反馈。

## 它不会做什么

- 不会替你定义“你是什么样的人”。
- 不会根据一次对话自动生成画像。
- 不会把推测直接写成稳定偏好。
- 不会在未确认时改变 Agent 的长期合作方式。

## 如何开始

```text
我想让 Agent 更好理解我的协作方式。请启用人物模块，并带我完成首次问卷。
```

问卷会一次问一个问题，用容易比较的选项帮助你表达倾向；你可以选“看情况”或不确定，不必强迫自己落进某个类型。

完成后会形成一份只属于当前工作副本的人物档案，并可派生出本地协作偏好。

## 如何校准

当你觉得当前档案不贴近实际时，可以说：

```text
请根据我最近真实的协作行为校准人物模块，先只给建议。
```

Agent 会先提出哪里可能不一致、依据是什么；只有你确认后才更新档案。人物模块的价值在于不断靠事实修正，而不是一次测试定终身。
